Details about interfaces

Details about main menu page

The games main menu will be as simple as it looks. I wanted something minimal, not too croweded. It includes a play button, which starts the game, an achievement button that shows a list of all achievements completed and not completed, three settings buttons, an exit button and a profile button.

Details about settings on main menu page

In case we don't won't our phone to vibrate during gaming we can just click the second out of the three settings button. We can see that something will be disabled if it's colored with red and enabled if it's colored in green.

Details about profile page

It includes 3 fields such as name, email and details, a profile picture, the level of the current user and two buttons, one for save and the other to get back to main menu. I added 2 more images regarding two problems I consider mandatory to be discussed. Operating the buttons I found the following problems: saved incorrectly (when not all mandatory fields are completed correctly), pressing back button without save (in this case the user must press the back button again, losing all the modifications, or he must save and get back to main menu).

Details about achievement list

This page includes all achievements you can possibly complete in this game. There will be two types of achievements: completed and not completed; each of them having different styles, colors, fields.

Details about button interaction

I feel like giving some diversity to the desing would be great so when a user interacts with a button it will have different effect (effects are being chosen from a list of available effects)
